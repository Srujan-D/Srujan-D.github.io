GenNav

GenNav, short for ‘General Navigation’, was undertaken by the Electronics and Robotics Club at BITS Goa. As we did the project during the COVID-19 lockdown, I remotely contributed to the open-source project.

GenNav is a python library for robotics navigation algorithms and utilities commonly required in navigation stacks. The library aims to be completely modular and have a unified API so that it is usable in a broad range of applications and easily extensible to new robotics domains. We further extended the project to construct a ROS Wrapper on top of Gennav, called ‘gennav_ros’. ‘gennav_ros’ can be used if someone wishes to use ‘gennav’ in a ROS based stack. Since the package is open-sourced using PyPi here, GenNav can also be installed using pip install gennav, although installation from source is recommended. Rudimentary documentation for the project can be found here.

As part of the team, I implemented path planning algorithms like Rapidly‐exploring Random Tree (RRT), Rapidly‐exploring Random Graph (RRG), and some utility classes and methods for implementing Graphs, etc. While implementing the algorithms from scratch was challenging (albeit fun), the more important thing to take care of was maintaining the API across all the path planners and making all generalised pieces of code ‘modular’. I also made Jupyter and Google Colab notebook tutorials for showing how to use the available algorithms. The notebooks can be found here.
